Buttons & Keys

Buttons Keys Action
0-9 Digit pressed will show up in the bar.
+ - * / . Basic operators (+ for adding, - for subtracting, * for multiplying, / for dividing, . for inserting a period i.e. for decimals).
_ (underscore) Pressing the button will negate the current value in the bar, if the bar contains an expression the button will insert a negative sign. Typing an underscore will also insert a negative sign.
'Enter' or 'return' = Evaluates current expression in bar, either by pressing the equals symbol or 'enter' on the keyboard.
None Toggles a popup for basic memory functions (MC for clear/back to 0, M+ for add, M- for subtract, MR for recall/print). For adding and subtracting, value must be in bar before pressing buttons.
V v Variable manager button. Toggles a popup for setting, removing, and using variables (first input with 'x' is for the variable name). Typing 'a' in the value input shows 'Ans()' where Ans(n) will give the variable the value of the nth answer entry. Once created, the variable will show up in the top left in subscript under V. Hovering over a variable will show the according value.
( Prints a bracket pair with the cursor position between them.
'Backspace' or 'delete' Removes the spaces before the cursor accordingly.
A Clears the bar.
p Pi or 3.141592653589793
e Euler's number or 2.718281828459046
! Factorial operator, allows decimals as well as integers.
% Percent operator, an equivalent shorthand to dividing by 100.
s c t Basic trigonometric functions (s for sin, c for cos, t for tan). This is when 2nd mode is turned off.
s c t Basic inverse trigonometric functions (s for sin-1, c for cos-1, t for tan-1). This is when 2nd mode is turned on.
C Combinations function which is equivalent to n! (n - r)!r! .
P Permutations function which is equivalent to n! (n - r)! .
^ Exponent operator.
S N The S (sum) function calculates the sum of the first n elements (value after 'S') of the set sequence. Holding down the button for 1 second will display a popup to customize the sequence. Pressing the button when the caret position is 1 ahead will toggle between S and N. The N (nth) function calculates the nth element as opposed to the sum of the first n elements.
l Logarithmic function with changeable base by holding down for 1 second. If the current set base is not 10, it will show up in the top left.
E Exponent of 10, equivalent to 10^.
L Natural logarithmic function equivalent to loge(x).
x Exponent of e, equivalent to e^.
r Square root function.
n Nth root function, will add 'n√()' to the bar so the root will have to be before adding. The value before the 'n' represents the root so 3n√(27) will evaluate 3 because that is the cube (third) root of 27.
@ Enables 2nd mode (visible in top left of bar).
None Toggles the 'angle' mode for the trigonmetric functions. Default is in degrees which can be toggled to radians. If the current mode is in radians, the result will show up in the top left of the bar.
None Toggles the evaluation precision. Holding down the 'exact' button for 1 second will give a popup to control how many decimal places to round to (between 1 and 9).
a References the last value calculated as a variable. Hold down for 1 second to manage data entry history.
'Space' Toggles the blinking cursor.
None Copies the current bar to clipboard.
None Pastes only valid expressions from clipboard. If invalid, a PasteError will be thrown.


'ArrowLeft' 'ArrowRight' Moves the current caret position either left or right accordingly.
